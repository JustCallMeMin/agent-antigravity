# Content Optimization & Structure

## 1. Intent-Centric Content
- **Information**: Clear answers, definitions, guides.
- **Transaction**: Optimized CTAs, product features, prices.
- **Comparison**: Tables, pros/cons, neutral analysis.

## 2. On-Page Structure
- **H1 Header**: Exactly one per page, containing the primary keyword/intent.
- **Hierarchy**: Use H2-H4 logically. Never skip levels.
- **URL Slugs**: Short, lowercase, hyphenated, descriptive.

## 3. Semantic Richness
- Use synonyms and related entities (LSI).
- Include FAQs to target "People Also Ask" snippets.
- Use internal linking to distribute "link juice" and help crawlers navigate.

## 4. Metadata Authority
- **Title Tag**: < 60 chars. Leading keyword. Compelling CTA.
- **Meta Description**: < 160 chars. Summarize the page and encourage the click.
- **Alt Text**: Descriptive for both accessibility and image search visibility.
