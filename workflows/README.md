# Workflows

This directory contains slash command definitions for user interaction.

## Slash Commands Reference

| Command | Description | Agent / Skill |
| :--- | :--- | :--- |
| `/brainstorm` | Explore ideas and options | `brainstorming` skill |
| `/create` | Initialize new application | `app-builder` skill |
| `/debug` | Root cause analysis & fixing | `systematic-debugging` skill |
| `/deploy` | Production deployment | `deployment-procedures` skill |
| `/enhance` | Add/update features | `orchestrator` agent |
| `/orchestrate` | Multi-agent coordination | `orchestrator` agent |
| `/plan` | Task planning (no code) | `plan-writing` skill |
| `/preview` | Local server management | `auto_preview.py` |
| `/status` | Project/Agent health check | `session_manager.py` |
| `/test` | Test generation & running | `testing-patterns` skill |
| `/ui-ux-pro-max` | High-end design system | `frontend-design` skill |
